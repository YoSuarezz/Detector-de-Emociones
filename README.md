Pasos para ejecutar el proyecto:
1. Despues de clonar el repositorio, crear el entorno de python con "py -3.10 -m venv .venv" y activarlo con ".venv\Scripts\Activate.ps1", se debe tener python 3.10.11
2. descargar los requerimientos con "pip install -r webui/requirements.txt" y "pip install -r requirements.txt"
3. Crear una carpeta data en la raiz del proyecto con los audios a probar en formato .wav
4. Ejecutar la aplicacion con python webui/app.py
